# Deloitte Job Simulation

A Python project completed as part of the Deloitte job simulation program. 
This repo processes input data (`data-1.json`, `data-2.json`) and generates 
a result output (`data-result.json`) using `main.py`.

## Files
- `main.py` — main script for data processing
- `data-1.json`, `data-2.json` — input data files
- `data-result.json` — output/result file
